# Gems related

## Gems

### Display gems infos
```shell
$ gem env
```

### Display gem location
```shell
$ bundle info GEM_NAME
```

### Update Gemfile.lock bundler
```shell
$ bundle update --bundler
```

### Remove all gems
```shell
$ gem uninstall -aIx
```
